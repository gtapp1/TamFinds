# Implementation Steps

## Phase 1: Foundation
1. Execute "Big Bang" command to install dependencies.
2. Initialize `src/api/firebaseConfig.ts`.
3. Create `src/theme/colors.ts` with FEU branding.
4. Build `useAuth` hook for flexible email login.

## Phase 2: Core Features
1. Build "Snap & Go" Reporting (Camera + Firestore Upload).
2. Build Home Feed (Real-time listener for Items).
3. Implement Item Detail view with "Contact Owner" trigger.

## Phase 3: Polish
1. Add FEU-themed loading skeletons.
2. Implement image compression before upload.
3. Final UI pass for FEU Roosevelt branding consistency.
